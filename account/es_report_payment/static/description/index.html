Key Features

  • Prints Mongolian Cash Receipt by Vendor Cash Payment;
  • Prints Mongolian Bank Receipt by Vendor Bank Payment;

Setup

Go to Accounting>Configuration>Settings menu.

Usage

Print Bank Receipt from Vendor Payment

Bank Receipt print button is visible when payment journal is related with Bank.

Bank Receipt Report View

Print Cash Receipt from Vendor Payment

Cash Receipt print button is visible when payment journal is related with Cash.

Cash Receipt Report View
